Configuración de seguridad para Apache HTTPD#

Cambio de directivas básicas 
<DirectoryMatch "/\.git">
   Require all denied
</DirectoryMatch>

Se establece el acceso para los directorios que sean extensiones de git, lo cual denegará el acceso a todos los usuarios, lo que nos sirve para evitar que algún usuario de manera intencional o no obtenga acceso a algún archivo de configuración o con dtos sensibles de nuestro sistema de versiones.

Se aplican los comandos

a2enconf security
apachectl -t
systemctl reload apache2
apachectl -S
